# analiseProblemas

Transferencia dos dados feitas na empresa para o R, para que mostre os tipos de problemas mais freguentes por mês(a análise já foi feita previamente pelo sheets usando função SQL).
Assim, programando para que de modo interativo o usuario escolha o mês para que seja mostrado quais os 5 maiores problemas por mês.De modo que, é possível também abrir o gráfico na janela do navegador(não como um sistema hospedado, mas entrando pelo ip host gerado pelo código).
